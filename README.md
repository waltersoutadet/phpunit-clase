# 🧪 PHPUnit Clase Demo

Proyecto base para aprender testing en PHP con PHPUnit.

## 🚀 Setup

```bash
composer install
``` id="setup1"

## ▶️ Ejecutar tests

```bash
composer test
``` id="setup2"

## ✅ CI (GitHub Actions)

Este proyecto corre tests automáticamente en cada push y pull request.

## 🎯 Objetivos

- Entender testing unitario
- Detectar bugs
- Escribir tests efectivos

## 🧩 Ejercicios

1. Hacer que todos los tests pasen
2. Agregar nuevos casos de prueba
3. Crear nuevas funcionalidades con tests

## 💡 Tip

Si el CI falla, tu código no está listo 😄
